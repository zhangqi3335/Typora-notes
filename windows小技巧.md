## 将.exe文件在windows搜索中找到

### 创建快捷方式放入“开始菜单”文件夹（最推荐，效果最好）

这是 Windows 的标准做法。这样做后，搜索结果会把它当做一个“应用程序”显示，看起来最整洁。

1. 找到你的 .exe 文件，右键点击 -> **创建快捷方式**。

2. （可选）把这个快捷方式重命名为你想要搜索的关键词（例如把 tool_v1.2.exe 的快捷方式改成 我的工具）。

3. 按下 Win + R 键，输入以下命令并回车：

   - 如果你只想让自己看到：输入 

     ```powershell
     shell:programs
     ```

   - 如果你想让电脑上所有用户都看到：输入 

     ```powershell
     shell:common programs
     ```

4. 这会打开一个文件夹。把刚才创建的**快捷方式**剪切/复制到这个文件夹里。

5. **完成。** 等几秒钟索引刷新后，你按 Win 键搜索这个名字，就能直接搜到了。
